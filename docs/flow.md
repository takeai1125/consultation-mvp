# 相談窓口サービス 全体設計

## 1. サービス利用フロー（Instagram → 相談 → 対応）

```mermaid
flowchart TD
    A[Instagram投稿を見る] --> B[Instagram DMを送る]
    B --> C[相談窓口サイトを案内]
    C --> D[トップページ<br/>注意書きを読む]

    D --> E[相談内容を入力]
    E --> F[相談を送信]

    F --> G[自動返信メッセージ表示<br/>送信完了ページ]

    G --> H{継続相談を希望？}
    H -- いいえ --> I[ここで終了]
    H -- はい --> J[対応時間の予約]

    J --> K[予約確定]
    K --> L[指定時間に相談対応]
